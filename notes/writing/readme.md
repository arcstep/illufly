# 生成流程
===================

## 生成所有章节清单
- 生成章节和摘要
- 每个章节一个文件

## 生成单个章节的小节结构
- 人工提供
- 头脑风暴
- 自动生成列表
- 允许在同一个文件中补充

## 扩展每个小节的要点
- 人工提供
- 头脑风暴
- 生成要点

## 根据要点扩展细节内容
- 重写
- 扩写
- 翻译
