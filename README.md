**langchain_chinese** 的目标是提供中文大语言模型和中文友好的`langchain`工具。

## 安装

你可以使用 pip 安装：
```
pip install -U langchain_chinese
```

或者使用 poetry 安装：
```
poetry add langchain_chinese
```

然后
